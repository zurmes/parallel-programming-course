Репозиторий содержит лабораторные работы по курсу «Параллельное программирование», посвященные различным технологиям параллельных вычислений на примере задачи умножения квадратных матриц.

## Аппаратные характеристики тестового стенда
*   **Процессор (CPU):** AMD Ryzen 5 6600H (6 физических ядер / 12 логических потоков).
*   **Видеокарта (GPU):** NVIDIA GeForce RTX 3050 Laptop GPU (4 GB, архитектура Ampere).
*   **Оперативная память (RAM):** 32 GB DDR5.
*   **Суперкомпьютерный кластер:** Высокопроизводительный кластер «Сергей Королёв».

## Содержание лабораторных работ

1.  **[Лабораторная работа №1: Последовательный алгоритм](./lab1/README.md)**  
    Реализация базового последовательного алгоритма умножения матриц со сложностью $O(N^3)$. Использование одномерных массивов для кэш-оптимизации. Создание системы верификации результатов через Python (NumPy).

2.  **[Лабораторная работа №2: Технология OpenMP](./lab2/README.md)**  
    Многопоточное распараллеливание алгоритма на процессорах с общей памятью. Оценка масштабируемости и построение графиков ускорения для конфигураций от 1 до 12 потоков.

3.  **[Лабораторная работа №3: Технология MPI](./lab3/README.md)**  
    Перенос вычислений на системы с распределенной памятью. Горизонтальная декомпозиция матриц с помощью коллективных операций `MPI_Bcast`, `MPI_Scatter` и `MPI_Gather`.

4.  **[Лабораторная работа №4: Вычисления на GPU (CUDA)](./lab4/README.md)**  
    Реализация оптимизированного тайлового алгоритма умножения матриц с использованием сверхбыстрой разделяемой памяти (Shared Memory). Оценка влияния размеров блока потоков на производительность GPU.

5.  **[Лабораторная работа №5: Суперкомпьютер «Сергей Королёв»](./lab5/README.md)**  
    Освоение работы на распределенном высокопроизводительном кластере. Настройка окружения компиляторов Intel, сборка проекта через `mpiicpc` и запуск пакетных заданий через планировщик Slurm.
